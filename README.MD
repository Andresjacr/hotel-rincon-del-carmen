# Hotel el Rincón del Carmen - Sitio Web

Un sitio web completo y funcional para el Hotel el Rincón del Carmen, desarrollado con tecnologías web modernas y un enfoque mobile-first.

## 🏨 Características Principales

### Para Clientes
- **Página de Inicio Atractiva**: Landing page con carrusel de habitaciones, áreas del hotel y servicios
- **Sistema de Reservas**: Búsqueda de disponibilidad, filtrado por fechas y huéspedes
- **Gestión de Cuenta**: Panel de usuario para ver y gestionar reservas
- **Sistema de Autenticación**: Registro e inicio de sesión seguro
- **Página de Contacto**: Formulario de contacto, información de ubicación y FAQ

### Para Administradores
- **Panel de Administración**: Gestión completa de habitaciones y reservas
- **Gestión de Usuarios**: Visualización de usuarios registrados
- **Gestión de Mensajes**: Revisión de mensajes de contacto
- **Estadísticas**: Dashboard con métricas del hotel

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica y accesible
- **CSS3**: Diseño responsivo con CSS Grid y Flexbox
- **JavaScript ES6+**: Lógica de aplicación y funcionalidades
- **Web Components**: Componentes reutilizables y modulares
- **Local Storage**: Simulación de base de datos para persistencia de datos

## 📱 Diseño Responsivo

El sitio web está diseñado con un enfoque **mobile-first**, asegurando una experiencia óptima en todos los dispositivos:

- **Móviles**: Navegación hamburguesa, diseño de columna única
- **Tablets**: Diseño adaptativo con grid responsivo
- **Desktop**: Layout completo con sidebar y múltiples columnas
